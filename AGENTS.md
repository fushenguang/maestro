# Maestro agent entrypoint

Start here when working in this repository:

- Reference guidance: [`docs/references/AGENTS.md`](./docs/references/AGENTS.md)
- OpenSpec workflow: [`openspec/AGENTS.md`](./openspec/AGENTS.md)
- Architecture charter: [`docs/architecture/vision.md`](./docs/architecture/vision.md)

To start a new change, create `openspec/changes/<change-id>/`, add `proposal.md`, `design.md`, `tasks.md`, and the matching `specs/<capability>/spec.md`, then run `openspec validate <change-id> --strict` before implementation.
