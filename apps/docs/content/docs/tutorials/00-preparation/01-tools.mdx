**配项目**：装好本章推荐的 3 个工具，用它们打开本教程的示例项目。

## 计划覆盖

- **VS Code**：本教程统一使用的编辑器（免费、跨平台、内置终端）
- **代码查看与 Diff 工具**：快速浏览 CC 改了什么
- **终端增强**：iTerm2（macOS）/ Windows Terminal / Warp——比原生终端好用
- **文档管理**：Markdown 预览、笔记工具推荐（Obsidian / Typora）
- **Git GUI**（可选）：Fork / Sourcetree——不记命令也能用 Git

## 为什么需要这些

Claude Code 在终端里跑，但你还需要看代码、对比改动、管理项目文档。工具搭配好了，CC 生成的代码你能更快理解和验证。
