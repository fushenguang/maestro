## 计划覆盖

- Hooks 是什么：CC 的操作拦截系统
- 5 类 handler：PreToolUse / PostToolUse / Notification / SessionStart / Stop
- 3 层事件模型：工具层、会话层、系统层
- Hooks 不是「限制 CC」——是「让 CC 在安全轨道上跑」
