## 计划覆盖

- 三种权限选择：allow / deny / always allow 的决策树
- 权限粒度：工具级 vs 路径级 vs 命令级
- 常见误用：全点 yes（不安全）vs 全点 no（用不了）
- 权限策略模板（个人开发 / 团队协作）
