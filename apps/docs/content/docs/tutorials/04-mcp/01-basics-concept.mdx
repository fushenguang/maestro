## 计划覆盖

- MCP 是什么：Model Context Protocol 的设计哲学
- 三层模型详解：工具（Tool）、资源（Resource）、提示（Prompt）
- MCP 不是「额外功能」——是 CC 的感官系统
- 常见 MCP Server 类型：数据库 / API / 文件系统 / 搜索
