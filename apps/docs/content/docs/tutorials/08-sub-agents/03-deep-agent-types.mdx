## 计划覆盖

- Explore agent：Haiku 驱动，只读搜索——最快但能力最窄
- Plan agent：计划模式下只读研究——避免与主循环互相递归
- general-purpose agent：继承主会话模型与工具——最全面但最贵
- 辅助 agent（statusline-setup / claude-code-guide）的使用场景
