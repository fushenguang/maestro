## 计划覆盖

- 并行 (parallel)：独立任务同时跑——Wall-clock = 最慢的那个
- Pipeline：任务有依赖关系——前一阶段的输出是后一阶段的输入
- Workflow 脚本：用 JavaScript 编排复杂的多 agent 流程
- 循环直到收敛：未知数量的发现任务——找到无新发现为止
- 常见错误：不需要 barrier 却加了——浪费等待时间
