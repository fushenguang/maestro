**项目产出**：一个可视化 Git 客户端——打开本地仓库、Monaco diff 双栏、Conventional Commits 模板、分支列表。

## 计划覆盖

- 打开本地 Git 仓库（自动识别 .git）
- git2-rs（libgit2 Rust 绑定）异步包装
- Monaco Editor diff 双栏 + 暂存/取消暂存
- Conventional Commits 模板 + 分支列表

## 最大坑（前置告知）

git2-rs 文档稀缺——主要靠源码和 GitHub Issues 学习。大仓库 diff 性能。SSH 密钥跨平台。
