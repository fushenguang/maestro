**项目产出**：一个付费 Newsletter 平台——Stripe 订阅、Magic Link 登录、付费墙、Resend 邮件发送。

## 计划覆盖

- Stripe Checkout 订阅（$5/月 或 $50/年）
- Stripe Webhook 本地调试（stripe listen --forward-to）
- Resend 邮件 + SPF/DKIM/DMARC
- Magic Link 登录 + 付费墙
- 中国大陆主体 Stripe 审核风险（3 种可行路径）

## 最大坑（前置告知）

Stripe 中国大陆主体被拒概率高。邮件进垃圾箱。最难的不是技术——是持续写 6 个月。
