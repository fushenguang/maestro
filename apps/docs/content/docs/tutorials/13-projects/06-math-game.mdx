**项目产出**：一个数学闯关游戏——题目生成器、闯关模式（10 题一关）、答题动画、错题本自动收集、家庭排行榜。

## 计划覆盖

- 题目生成器（加减乘除，按年级分档）
- 闯关模式（错 3 题 game over）
- 答题动画（撒花/红叉）
- 题目约束：不能出负数、不能除以 0、不能太难

## 最大坑（前置告知）

题目生成器约束条件多。UI 配色/动效要花心思。避免变成纯游戏——不能跳过题目直接解锁。
