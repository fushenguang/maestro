**项目产出**：一个本地 Markdown 笔记应用——双链解析、FTS5 中文全文搜索、三栏布局、文件系统监听。

## 计划覆盖

- 文件系统监听（notify crate）
- 双链解析（`[[文件名]]` + 反向链接）
- FTS5 中文分词（jieba-rs）
- 三栏布局（文件树 + CodeMirror 6 + Marked.js）

## 最大坑（前置告知）

FTS5 默认对中文不友好。文件系统监听跨平台差异（FSEvents / ReadDirectoryChangesW / inotify）。
