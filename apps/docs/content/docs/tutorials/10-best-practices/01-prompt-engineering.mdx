**配项目**：重构一个模糊 prompt（「帮我写个网站」→ 精确的、带上下文的分步 prompt），对比 CC 的产出质量。

## 计划覆盖

- Prompt 三原则：精确 > 含糊、上下文 > 裸问、分步 > 一口气
- 反模式案例集：「帮我写个网站」「修一下这个 bug」「优化性能」——为什么这些是灾难
- Prompt 模板：功能开发 / Bug 修复 / 代码审查 / 项目初始化
- 自检：把你的常用 prompt 贴给 CC，让它审一遍
