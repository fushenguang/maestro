**配项目**：对你的 CC 使用习惯做一次审计——找出你在用的反模式，逐个修复。

## 计划覆盖

- 权限反模式：全点 yes vs 全点 no
- Hook 反模式：0 个 Hook 裸奔（「我从来不用 Hook」）
- Session 反模式：一个 session 做一切（不 compact、不分段）
- Skill 反模式：太模糊 / 太死 / 太长
- Prompt 反模式：太含糊 / 无上下文 / 一口气丢 10 个需求
- Memory 反模式：CLAUDE.md 写成文档而不是速查卡
