**配项目**：贪吃蛇项目同时开 3 个 worktree 并行开发 3 个功能——规范 commit、合入 PR。

## 计划覆盖

- Worktree 隔离：一个 worktree 一个功能，CC 在里面独立工作
- Conventional Commits：feat/fix/chore——让 commit message 不再是「update」
- PR 合入：squash merge vs rebase merge 的场景选择
- 自检：开两个 worktree，同时改贪吃蛇两个功能，互不影响
