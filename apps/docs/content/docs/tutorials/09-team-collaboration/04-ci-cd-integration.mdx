**配项目**：给团队 CI 加一个「自动修 bug」步骤——测试挂了 → CC 分析错误日志 → 生成修复 → 自动提 PR。

## 计划覆盖

- 3 种集成模式：CLI 调用 / SDK 嵌入 / API 直调
- CI/CD 中的 CC：不是替代 CI，是补充 CI 做不到的事
- 自动修 bug 流水线：测试失败 → CC 分析 → 修复 → 验证 → PR
- 安全边界：CI 中的 CC 权限必须比人用的更严格
