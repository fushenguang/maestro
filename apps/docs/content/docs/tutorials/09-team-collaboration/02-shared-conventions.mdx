**配项目**：为一个团队仓库建立共享 CLAUDE.md + 3 个核心 Skill + Memory 策略——新成员第一天就能用 CC 按团队规范写代码。

## 计划覆盖

- 共享 CLAUDE.md：Git 版本控制 + Code Review 流程
- 共享 Skill 库：团队级 Skill（Code Review / Deploy / Onboarding）
- Memory 同步策略：哪些 Memory 该共享、哪些该个人
- 规范演进：CC 的使用规范也是代码——PR + review + merge
