**配项目**：给团队仓库配一个 PR Review Bot——每个 PR 自动跑 CC code-review，人工只看 AI 标记的「需要确认」项。

## 计划覆盖

- AI Code Review 的定位：第一遍筛选，不是最终裁决
- CC code-review 的配置：review 维度（安全/性能/风格/逻辑）
- 实战：写一个 PR Review Skill——自动检查 + 按严重度分类
- 人机分工：什么错误 AI 可以直接修、什么必须人来判断
