# gemini-cli

## 一句话定位

> Google 开源 Gemini 终端 coding agent：TS + Ink + 7 包 monorepo，Gemini3 + 1M context + MCP + Hooks + Skills + Subagents。

## 关键 takeaway

1. **栈与包**：Node >= 20 + TS + React Ink；7 包 monorepo：`cli` / `core` / `a2a-server`(A2A) / `sdk` / `devtools` / `test-utils` / `vscode-ide-companion`。(methodology:L21)
2. **三认证**：Google OAuth 免费 60 req/min + Gemini3 + 1M / `GEMINI_API_KEY` 1k/day / Vertex AI。(methodology:L150-L209)
3. **三层 GEMINI.md**：全局 → workspace → JIT 工具访问时向上扫到 trusted root。(methodology:L11-L37)
4. **MCP + Hooks**：stdio / SSE / Streamable HTTP + 独立 Hooks。(methodology:L29-L60)
5. **差异化**：Gemini3 + 1M context；multimodal（PDF / image / sketch → app）；Google Search grounding；Checkpointing + `/rewind`；Plan mode；Headless + `stream-json`。
