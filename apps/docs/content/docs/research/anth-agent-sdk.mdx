# Claude Agent SDK（官方文档）

## 一句话定位

> Agent SDK 把 Claude Code 整套能力（工具执行、agent loop、context 管理）封装成 Python / TypeScript 库，让你在自己的进程 + 自己的基础设施里跑生产 agent，无需重新实现 tool 循环。

## 关键 takeaway

1. **三条 agent 路径**：Client SDK（你自己写 tool loop）vs **Agent SDK（库 + 你的进程 + 你的基础设施）** vs Managed Agents（Anthropic 托管 REST + 沙箱）。典型路径：Agent SDK 原型 → Managed Agents 生产。锚点 `docs:L1`。
2. **双语言对称**：`npm install @anthropic-ai/claude-agent-sdk`（TS，**捆绑平台原生 Claude Code binary** 作为 optional dependency）vs `pip install claude-agent-sdk`（Python 3.10+）；API 平行，示例可一一对照。锚点 `docs:L1`（"The TypeScript SDK bundles a native Claude Code binary"）。
3. **认证 5 种**：`ANTHROPIC_API_KEY` + 4 个第三方 provider 经环境变量开关——`CLAUDE_CODE_USE_BEDROCK=1` / `CLAUDE_CODE_USE_ANTHROPIC_AWS=1`（+ `ANTHROPIC_AWS_WORKSPACE_ID`）/ `CLAUDE_CODE_USE_VERTEX=1` / `CLAUDE_CODE_USE_FOUNDRY=1`。
4. **开箱工具 10 个 + 6 类 hook + Subagent + MCP**：Read / Write / Edit / Bash / **Monitor**（后台脚本按行事件流，新工具）/ Glob / Grep / WebSearch / WebFetch / AskUserQuestion；hook 类型 `PreToolUse` / `PostToolUse` / `Stop` / `SessionStart` / `SessionEnd` / `UserPromptSubmit` 经 `HookMatcher(matcher="Edit|Write", hooks=[...])` 匹配；Subagent 委派经 `AgentDefinition(description, prompt, tools)` 注册 + `parent_tool_use_id` 字段追踪；MCP integration 一行 `mcp_servers: {"playwright": {"command": "npx", "args": ["@playwright/mcp@latest"]}}` 接外部 server。
5. **复用 Claude Code 配置 + session 连续性 + 订阅变更**：默认从 `.claude/`（项目）和 `~/.claude/`（用户）加载 Skills / Commands / Memory / Plugins，可用 `setting_sources`（Python）/ `settingSources`（TS）限制；`session_id` + `ClaudeAgentOptions(resume=session_id)` 跨查询 context 延续或 fork。**重要**：自 2026-06-15 起，Agent SDK 与 `claude -p` 在订阅 plan 上扣新的「Agent SDK credit」，独立于交互用量额度。锚点 `docs:L1`（"Starting June 15, 2026..."）。

## Brief（梳理）

### Agent SDK 定位：库 + 你的进程

- **核心抽象**：把 Claude Code 整套能力（工具执行 / agent loop / context 管理）封装成 Python / TypeScript 库
- **与 Claude Code CLI 关系**：CLI = 命令行调用；Agent SDK = 编程化嵌入到你自己进程
- **优势**：**不重新实现 tool 循环**——SDK 已含 Read/Write/Bash/Edit/MCP 调用机制

### 三条 Agent 路径（取舍决策表）

| 路径 | 抽象度 | 你负责 | 适合 |
|---|---|---|---|
| **Client SDK** | 最低 | 你自己写 tool loop / context 管理 | 完全自控 / 极简 |
| **Agent SDK** | 中 | 库 + 你的进程 + 你的基础设施 | **典型路径**：原型 → 生产 |
| **Managed Agents** | 最高 | Anthropic 托管 REST + 沙箱 | 不想运维基础设施 |

- **典型路径**：**Agent SDK 原型**（开发灵活）→ **Managed Agents 生产**（运维 0 负担）

### 双语言对称

| 语言 | 安装 | 备注 |
|---|---|---|
| **TypeScript** | `npm install @anthropic-ai/claude-agent-sdk` | **捆绑平台原生 Claude Code binary** 作为 optional dependency |
| **Python** | `pip install claude-agent-sdk`（Python 3.10+）| 同等 API |

- **API 平行**——TS / Python 文档示例可一一对照

### 5 种认证（环境变量开关）

| 认证 | 开关 |
|---|---|
| **Anthropic API** | `ANTHROPIC_API_KEY` |
| **AWS Bedrock** | `CLAUDE_CODE_USE_BEDROCK=1` |
| **Anthropic on AWS** | `CLAUDE_CODE_USE_ANTHROPIC_AWS=1` + `ANTHROPIC_AWS_WORKSPACE_ID` |
| **GCP Vertex** | `CLAUDE_CODE_USE_VERTEX=1` |
| **Foundry** | `CLAUDE_CODE_USE_FOUNDRY=1` |

### 开箱工具 10 个

- **基础 5**：Read / Write / Edit / Bash / Glob
- **搜索 2**：Grep / WebSearch
- **网络 1**：WebFetch
- **交互 1**：AskUserQuestion
- **新增 1**：**Monitor**（后台脚本按行事件流，新工具，区分于 Bash 长进程）

### 6 类 Hook + Subagent + MCP 扩展

- **6 类 hook**：`PreToolUse` / `PostToolUse` / `Stop` / `SessionStart` / `SessionEnd` / `UserPromptSubmit`
- **HookMatcher 模式**：`HookMatcher(matcher="Edit|Write", hooks=[...])` 按工具名正则匹配
- **Subagent 委派**：`AgentDefinition(description, prompt, tools)` 注册；**`parent_tool_use_id` 字段追踪**消息来源
- **MCP 一行接入**：`mcp_servers: {"playwright": {"command": "npx", "args": ["@playwright/mcp@latest"]}}`

### 配置加载 + Session 连续性

- **默认从 `.claude/`（项目）和 `~/.claude/`（用户）加载**：Skills / Commands / Memory / Plugins
- **限制加载**：`setting_sources`（Python）/ `settingSources`（TS）
- **Session 连续性**：`session_id` + `ClaudeAgentOptions(resume=session_id)` 跨查询 context 延续或 fork

### 商业化节点（2026-06-15）

- **新 Agent SDK credit 配额**（独立于交互用量额度）
- 影响范围：Agent SDK + `claude -p`（CLI 非交互模式）
- 触发条件：订阅 plan（Max / Pro / Team）
- **影响**：生产 agent 部署需评估 credit 用量，避免月底超额

### 品牌约束（重要）

- ✅ **允许**：「Claude Agent」（preferred）
- ❌ **禁用**：「Claude Code」/ 「Claude Code Agent」/ ASCII art
- **场景**：SDK 集成到产品时，UI / 文档 / marketing 措辞需遵守
