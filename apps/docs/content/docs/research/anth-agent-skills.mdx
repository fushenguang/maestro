# Agent Skills（官方文档）

## 一句话定位

> Anthropic 官方对 Skills 的定义、仓库结构与 SKILL.md 写法权威说明。

## 关键 takeaway

1. Skills 是「指令 + 脚本 + 资源」的目录，Claude 按需动态加载以提升专门任务表现（`README.md:L6`）。
2. 每个 skill 自包含在一个文件夹里，核心是 `SKILL.md`（frontmatter + instructions）（`README.md:L18`）。
3. 最小 `SKILL.md` 只需 `name` 和 `description` 两个 frontmatter 字段（`README.md:L78-81`）。
4. 仓库分三类：`./skills` 示例、`./spec` 规范、`./template` 模板（`README.md:L27-29`）。
5. 接入途径：Claude Code plugin marketplace、Claude.ai 付费计划、Claude API Skills endpoint（`README.md:L33-61`）。

## 与 Claude Code 教程的关联

- Claude Code 教程的 `/plugin marketplace add anthropics/skills` 命令直接对应本文「Try in Claude Code」段落（`README.md:L36`）。
- `SKILL.md` 的 frontmatter schema 是本仓库其他 Claude Code skill（如 `.claude/skills/`）的事实标准，可作为本项目 `research-source` skill 的对照参考。
