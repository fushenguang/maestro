# alirezarezvani/claude-skills

## 一句话定位

> 338+ 个 stdlib-only Python 工具 + SKILL.md 包，**一套代码 13 个 coding agent 通用**（CC / Codex / Gemini CLI / Cursor / Aider / Kilo / Windsurf / OpenCode / Augment / Antigravity / Hermes / Mistral Vibe / OpenClaw），靠工程治理而非教学叙事取胜。

## 关键 takeaway

1. **「一套代码 13 平台」统一适配**：`./scripts/convert.sh --tool all` 在 ~15 秒内把所有 skill 转为各平台原生格式（Cursor `.mdc` / Aider `CONVENTIONS.md` / Kilo `.kilocode/rules/` / Windsurf `.windsurf/skills/` / OpenCode `.opencode/skills/` / Augment `.augment/rules/` / Hermes `~/.hermes/skills/` / Vibe `~/.vibe/skills/` 等），这是其他源完全没占的格局（`README.md:109-148`）。
2. **「Algorithm over AI」+ stdlib-only 原则**：548+ Python 工具**全部标准库实现、零 pip install、不调 LLM**，跑得快、易 portable、可在 CI 跑（`CLAUDE.md` Key Principles #3、Anti-Patterns "LLM calls in scripts (defeats portability and speed)"）。
3. **企业级工程治理 + CI 验证**：`plugin.json` schema 强制 + `scripts/check_plugin_json.py --all` 在 `ci-quality-gate.yml` 每 PR 跑 + `skill-security-auditor` 安装前扫 6 大风险（command injection / code execution / data exfil / prompt injection / supply chain / privilege escalation）+ 分支保护（feature → dev → main，main 禁直推）（`CLAUDE.md` ClawHub Publishing Constraints + Git Workflow）。
4. **Path-B 11-file contract 标准化**：每 skill 必有 = 3 stdlib tools + 3 references × 5-7 sources + 1 template + cs-* agent + /cs:* command + plugin.json + SKILL.md；megaprompt 原文 verbatim 保留。**这是 superpowers「skill = SKILL.md only」的下一步演化**（`CLAUDE.md` v2.7.0 Highlights + 16 domains × 该模板）。
5. **16-17 个 domain 横向覆盖**：engineering / product / marketing / c-level / finance / RA-QM / research / research-ops / business-ops / commercial / markdown-html 等，比单工具源广度大 1 个量级；权威溯源密度极高（每 reference 5-7 个 source，引 DORA / Tufte / Cohen / Schoenfeld / NN/g / Kotler / IAS 38 等学术与行业经典）。
