## 简介

本目录是「Claude Code 教程」的**研究素材库**。每一个资料源（GitHub repo / 文章 / 教程 / 课程）都按以下三层逐级分析：

| Tier | 定位 | 字数（实测基线 / 按 source_subtype 分档）|
|---|---|---|
| **Card** | 速览：一句话定位 + 3-5 条 takeaway + 与教程的关联 | blog 1500 / article 1800 / docs 2500 / repo 1800 / cookbook 2000 / methodology 1800 / skill_collection 2000 |
| **Brief** | 梳理：章节结构 + 关键示例 + 跨源关联 | blog 3000 / article 3000 / docs 3500 / repo 3000 / cookbook 3500 / methodology 3000 / skill_collection 3500 |
| **Deep-Dive** | 深读：完整分析 + 教程章节候选 | ≤ 8000 字符（所有 subtype） |

当前阶段：**Gate 1（Card）**——先把所有候选源都做成 Card，再由人类 review 决定哪些升级到 Brief / Deep-Dive。

## 受众标签（三轴）

每个源按以下三轴打标签，读者可按自己的画像挑选阅读：

- **CS 背景**：`yes`（专业开发者）/ `partial`（会写但不深）/ `no`（不写代码）
- **DevOps 熟练度**：`low`（少用命令行）/ `mid`（日常 git + shell）/ `high`（CI/CD、容器、生产环境）
- **Claude Code 经验**：`newbie`（首次接触）/ `used`（用过几次）/ `advanced`（写自定义 Skill / Plugin）
